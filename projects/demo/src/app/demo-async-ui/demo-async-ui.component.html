SoftAsyncUI

softBusy

  • - loading state apply to container or page
  • - give a subscription or boolean to determine loading state

Long loading content


Long loading content (no backdrop) 

      # html
      <div [softBusy]="busy" class="is-busy-parent">
        Long loading content
      </div>
      
      <div [softBusy]="busy" [busyClass]="'no-backdrop'" class="is-busy-parent">
        Long loading content (no backdrop)
      </div>

      # ts
      this.busy = of().pipe(delay(Number.MAX_SAFE_INTEGER)).subscribe();

softLoading & softDisabled

  • - loading/disabled state apply to element e.g. button
  • - give a subscription or promise or boolean to determine loading/disabled state
      # html
      <button class="button"
        [softLoading]="loadingSub"
        (click)="testLoading()">
        Click here
      </button>

      <button class="button"
        [softLoading]="loadingSub"
        (click)="testLoading()">
        Click another
      </button>

      <button class="button"
        [softDisabled]="loadingSub">
        Just Disabled
      </button>

      # ts
      testLoading() {
        this.loadingSub = of().pipe(delay(3000)).subscribe();
      }
