SoftPipe

safe pipe

  • - 'html' trust html content e.g. [innerHTML]
  • - 'style' trust css content e.g. [style.background-image]
  • - 'script' trust javascript content e.g. content with <script>...</script>
  • - 'url' trust a value that can be used in hyperlinks e.g. <img src>
  • - 'resourceUrl' trust a location that may be used to load executable code from, e.g. <script src> <iframe src> 
      <iframe [src]="'https://softever.co.th' | safe: 'resourceUrl'"></iframe>
